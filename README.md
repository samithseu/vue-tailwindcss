# Vue + Tailwind CSS

This template created for using Vue + TailwindCSS project instantly by cloning this project using below command:

PowerShell:

```powershell
git clone https://github.com/samithseu/vue-tailwindcss.git; cd vue-tailwindcss
```

Bash:

```shell
git clone https://github.com/samithseu/vue-tailwindcss.git && cd vue-tailwindcss
```
